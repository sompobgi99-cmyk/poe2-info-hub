## ⚔️ Warrior

> 🎯 **Ascendancy แนะนำ:** <img src="https://assets-ng.maxroll.gg/images/poe2/ascendancy/titan.webp" width="22"> Titan (Warbringer ก็ใช้ guide เดียวกันได้)

### 📋 Skill Progression

| Act | Level | สกิลหลัก | Support แนะนำ |
|-----|-------|----------|---------------|
| Act 1 | 1-15 | <img src="https://www.poe2wiki.net/images/a/a9/Rolling_Slam_inventory_icon.png" width="20"> **Rolling Slam** → <img src="https://www.poe2wiki.net/images/8/83/Earthquake_inventory_icon.png" width="20"> **Earthquake** (lv12) | Fist of War, Brutality |
| Act 2 | 15-25 | <img src="https://www.poe2wiki.net/images/8/83/Earthquake_inventory_icon.png" width="20"> **Earthquake** + <img src="https://www.poe2wiki.net/images/1/15/Boneshatter_inventory_icon.png" width="20"> **Boneshatter** (stun) | Melee Phys, Heavy Swing |
| Act 3 | 25-35 | <img src="https://www.poe2wiki.net/images/1/15/Boneshatter_inventory_icon.png" width="20"> **Boneshatter** (main) + **Supercharged Slam** | Pulverise, Rage |
| Cruel 1 | 35-45 | เพิ่ม **Ancestral Warrior Totem** | Ancestral Bond (Keystone) |
| Cruel 2 | 45-55 | Totem + Slam hybrid | Multiple Totems |
| Cruel 3 | 55+ | ครบ build endgame | — |

### 🗺️ Passive Tree Path (30 ข้อแรก)

```
→ Melee Damage (start)
→ Resolute Technique (Keystone — ห้ามพลาด!)
→ Life / Armour nodes (ล่างซ้าย)
→ 2H Mace Damage cluster
→ Stun Threshold reduction
→ Ancestral Bond (lv45+ — ตอนใช้ Totem)
```

### ⚡ Key Powerspikes

| Level | Powerspike |
|-------|-----------|
| **12** | Earthquake ปลดล็อค — clear เร็วขึ้น 2-3x |
| **16** | Boneshatter — stun chain = oneshot pack |
| **45** | Ancestral Bond Keystone — Totem 2x |
| **55+** | Facebreaker Unique (ถ้าหาได้) |

### 🛡️ Gear Priority

1. **2H Mace** — Physical DPS > ทุกอย่าง (vendor recipe: Rustic Sash + Whetstone + Mace)
2. **Movement Speed Boots** — 10%+ = ชีวิตดี
3. **Life + Armour** — ทุกชิ้น
4. **Resistance Rings** — Fire/Cold/Lightning cap (Act 3+)

### ⚠️ คำเตือน

- Warrior Act 1 **ช้าที่สุดในเกม** — อย่าท้อ!
- Rolling Slam แรกๆ กดแล้วรู้สึกหน่วง — 0.5.0 +1s atk time ทำให้ดีขึ้น
- **อย่าใช้ Shield skills** เป็น main damage — 0.5.0 โดน nerf หมด

## 🔑 Universal Leveling Tips

### 🛒 Vendor Recipes (ห้ามลืม!)

| Recipe | วัตถุดิบ | ผลลัพธ์ |
|--------|---------|---------|
| **Phys Weapon** | Rustic Sash (blue+) + Whetstone + Weapon | เพิ่ม % Physical Damage |
| **+1 Spell Gems Wand** | Wand + Ruby Ring (blue+) + Alteration Orb | +1 Level of Spell Gems |
| **Movement Speed Boots** | Boots + Quicksilver Flask + Augmentation | 10% Movement Speed |
| **Resistance Ring** | Ring + skill gem ตามธาตุ | Fire/Cold/Lightning Resistance |

### 🏃 Speed Tips

1. **Movement Speed Boots** — สำคัญที่สุด! เช็ค vendor ทุกระดับ
2. **Quicksilver Flask** — Act 1 quest reward
3. **Skip mobs** — ฆ่าเฉพาะ blue pack + rare = XP เพียงพอ
4. **รู้ทางลัด** — Act 2 Dreadnaught สั้นลง 0.5.0!
5. **Act 3** — Waterways ไม่ต้องกด lever แล้ว → เดินผ่านเลย

### ⚡ Powerspike ระดับสากล

| Level | สำคัญ |
|-------|-------|
| **8** | Support gem แรกปลดล็อค |
| **22** | Ascendancy แรก (Trial of Sekhemas) |
| **28** | Gems ใหม่แรงๆ (Comet, Charged Staff, Cull the Weak) |
| **31** | Supports ระดับ 2 (Chain, LMP, Feeding Frenzy) |
| **45** | Supports ระดับ 3 (Rage, Trinity) |
| **55** | CI transition / Endgame prep |

### 💀 Hardcore Tips

- **Act 1 boss** (The Executioner) — หลบ slam ด้านข้าง
- **Act 2 boss** (Jamanra) — Lightning res cap!
- **Act 3 boss** (Doryani) — Chaos damage — Amethyst Flask
- <img src="https://assets-ng.maxroll.gg/images/poe2/ascendancy/chronomancer.webp" width="22"> **Chronomancer** = HC god tier (30% less damage taken)
