## 1. ✅ Confirmed Patch Changes (🟢 Official — ยืนยันจาก GGG)

### Chronomancer — 🔥 Rework 5 จุด

| เปลี่ยน | ผล |
|--------|-----|
| **Phased Form** — ลด damage 30% + delay 4s | กัน one-shot |
| **Temporal Rift** — cast 0.1s + ล้าง delayed damage | panic button |
| **Now and Again** — Echo/Repeat 20% | damage เพิ่ม |
| **Quicksand Hourglass** — 1-60% Skill Speed | เร็วทุกอย่าง |
| **Inevitable Agony** — Life Loss + auto Cull | rework ใหม่ |

### Pathfinder — 2 nerfs

| เปลี่ยน | ผล |
|--------|-----|
| Overwhelming Toxicity: less Poison Duration 35→**50%** | Poison หมดเร็ว |
| Running Assault: less Movement Penalty 50→**30%** | ติด penalty ง่ายขึ้น |

### Blood Mage — Leech node rework

| เปลี่ยน | ผล |
|--------|-----|
| Vitality Siphon: 10→**20%** Spell Leech | Buff |
| Flask ไม่ recover Life (100% less) | 🔴 ต้องทดสอบ |
| ⚡ **Leech ระบบเปลี่ยน:** single strongest instance + cap 40k dmg | ไม่ใช่บัฟล้วน |

### Other Confirmed Changes

| เปลี่ยน | ประเภท |
|--------|--------|
| **Leech** — single instance, cap 40,000, Vaal Pact ไม่ instant | 🔄 System rework |
| **Command Skills** — movement penalty 70→50% (Minion command เท่านั้น) | 🟢 System |
| **Banners** — ไม่มี movement penalty + radius 4.5→6m | 🟢 System |
| **Barkskin** — 50→62% ES as Armour | 🟢 System |
| **Comet** — damage -5% / Fire-Infused -25% | 🟡 Nerf |
| **Cull the Weak** — 109-281% dmg, Can't be Evaded (⚠️ Spear only) | 🟢 Buff |
| **Tempest Bell** — max 3 bells + Ancestrally Boosted | 🟢 Buff |
| **Culling Threshold** — 30→35% | 🟢 Monster |
| **Lightning Arrow** — beams ไม่ chain ซ้ำ target | 🔴 Nerf |
| **Ice Shot** — เสีย 25% more Freeze Buildup | 🟡 Nerf |
| **Minion damage bug fix** — +25-35% late-game | 🟢 Fix |
| **Companion 19 passive nodes** — global tree (ทุกคลาสใช้ได้) | 🌳 System |

## 2. 📈 Expected Winners (🟡 Prediction — วิเคราะห์ว่าใครน่าจะได้ประโยชน์สูง)

### Titan — ได้ประโยชน์สูงจากระบบใหม่

Titan **ไม่มี 🔴 buff โดยตรง** — แต่ระบบใหม่เอื้อ Melee หนักกว่าใคร:

| หมวด | เปลี่ยน | ผล |
|------|--------|-----|
| 🟡 Keystone | **Ancestral Bond** — Totem Limit x2, 75 Spirit | Totem build ใหม่ |
| 🟡 Keystone | **Vaal Pact** — 50% more Leech | Sustain |
| 🟢 System | Banners, Barkskin, Leech, Campaign | ทุก Melee ได้ |
| 💎 Unique | Facebreaker — Unarmed ใหม่ | Titan STR สูง = synergy |

> ⚠️ Warbringer ได้ของเหมือนกันทุกอย่าง — Titan ชนะที่ base ascendancy

### Deadeye — ระบบเอื้อ clear speed

| หมวด | เปลี่ยน |
|------|--------|
| 🟢 | Culling 30→35% + Leech + Campaign เร็ว |
| 🔴 | Mirage Deadeye ใช้ Channelled ได้ |
| 🔴 | LA no chain + Ice Shot freeze nerf |

### Amazon — Cull the Weak Spear buff

| หมวด | เปลี่ยน |
|------|--------|
| 🟢 | Cull the Weak rework — Spear skill, 109-281% dmg |
