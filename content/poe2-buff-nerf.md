## 🧑 Player Changes (ทุกคนได้)

| เปลี่ยน | สถานะ | กระทบหลัก |
|--------|--------|-----------|
| Bleed บนผู้เล่นไม่เพิ่ม damage ขณะเคลื่อนที่ | 🟢 Buff | Melee ทุกสาย, Titan, Invoker |
| Leech — instance เดียว, recovery สูงสุดชนะ | 🔄 Rework | Blood Mage, Acolyte, Titan |
| Leech cap — Hit >40,000 dmg = cap 40,000 | 🔴 Nerf | Blood Mage, Gemling |
| Deflect สูตรใหม่, cap 95% | 🟢 Buff | Invoker, Shield builds, Deadeye |
| Melee Splash ใช้ได้ครั้งเดียวต่อ area | 🔴 Nerf | Titan, Warbringer, Martial Artist |
| Parry/Shield Block ไม่ delay Heavy Stun | 🟢 Fix | Warbringer, Titan, Shield builds |
| Minion removal — ตัวใหม่ถูกลบก่อน | 🟢 QoL | Infernalist, Lich, Shaman, Spirit Walker |
| Archon Buffs ไม่หายเมื่อโจมตี | 🟢 Buff | Invoker, Martial Artist |
| Party revive หลังฆ่าบอส | 🟢 QoL | Party play, Tactician |
| Runic Ward เพิ่ม Honour ใน Sekhemas | 🟢 Buff | ทุก build ที่ใช้ Runic Ward |
| Sceptre/Allies mods ไม่ stack ใน party | 🔴 Nerf | Infernalist, Spirit Walker |
