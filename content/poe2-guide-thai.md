## 1. ภาพรวม

| หัวข้อ | รายละเอียด |
|--------|------------|
| **ชื่อเกม** | Path of Exile 2 |
| **ผู้พัฒนา** | Grinding Gear Games (GGG) — นิวซีแลนด์ |
| **แนวเกม** | Action RPG (ARPG) แนวดาร์กแฟนตาซี |
| **สถานะ** | Early Access (ตั้งแต่ 6 ธ.ค. 2024) |
| **แพตช์ล่าสุด** | **0.5.0 — Return of the Ancients** (29 พ.ค. 2026) |
| **แพลตฟอร์ม** | PC, PlayStation 5, Xbox Series X\|S |
| **ราคา Early Access** | ~$30 USD (Supporter Pack — มูลค่าเท่ากับ Points ในเกม) |
| **Full Release** | ฟรี (Free-to-Play) ภายในปี 2026 |
| **กำหนด 1.0** | หลัง ExileCon พ.ย. 2026 |

### ความแตกต่างจาก PoE 1

- **ต่อสู้ช้าลงแต่มีน้ำหนัก** — ไม่ใช่การกดปุ่มเดียวล้างจอ
- **WASD movement** — เคลื่อนที่ด้วย WASD ได้
- **12 คลาสตัวละคร** (จาก 7 ใน PoE 1)
- **ระบบ Skill Gem ใหม่** — สกิลลิงก์โดยตรง ไม่ต้องลิงก์ซ็อกเก็ตที่อุปกรณ์
- **Passive Tree ใหญ่กว่าเดิม** — >1,500 โหนด
- **Dodge Roll** — ทุกตัวละครกลิ้งหลบได้
- **Gold** — สกุลเงินใหม่สำหรับ Respec และ Gambling
- **Spirit** — ทรัพยากรสำหรับ Auras, Minions, Buffs ถาวร

## 6. ระบบ Endgame (รีเวิร์ค 0.5)

### Atlas of Worlds — ออกแบบใหม่หมด

| ระบบเดิม | ระบบใหม่ (0.5) |
|----------|----------------|
| Atlas สุ่ม | โซนเริ่มต้นมี League Mechanics **ประจำตำแหน่ง** |
| Atlas points จาก random maps | Atlas points จาก **Fortress** (เควสไลน์) |
| ต้อง respec | ปลดล็อก **ทั้งต้น** (300+ nodes) |
| Arbiter of Ash = จบ | **Arbiter of Divinity** ใหม่ + Pinnacle ครบทุก League |

### 🏰 Fortress — หัวใจของ Endgame ใหม่

```
เปิด Tower แรก → Fortress ผุดขึ้นกลาง Atlas
└── Maps ใน Fortress → ได้ Atlas Passive Points
    ├── Gateway Maps (3 แห่ง) — บอสใหม่ 2 ตัว
    ├── Citadel Maps (2 แห่ง) — บอสใหม่ + key สู่ Arbiter of Divinity
    └── Enigma Chambers (2 แห่ง) — Crisis Fragments แบบ Quest
```

### 🎓 Masters of the Atlas (ใหม่!)

ระบบ Ascendancy สำหรับ Endgame — ฝึกกับ Masters 3 คน:

| Master | สาย | ปลดล็อกโดย |
|--------|-----|-----------|
| **Doryani's Science** | Corruption | เคลียร์ Corruption Nexus |
| **Hilda's Hunting** | Monster Hunting | ไป Hilda's Campsite |
| **Jado's Spycraft** | Espionage | ทำ Anomaly Map |

- แต่ละ Master มี 12 nodes → เลือกได้ 4
- สลับ Master ได้ก่อนเข้า Map ทุกครั้ง
- โบนัสแบบ asymmetric — สร้าง build-defining choices

### 👑 Pinnacle Bosses ทั้งหมด

| บอส | League | วิธีเข้าถึง |
|-----|--------|-----------|
| **Arbiter of Divinity** 🆕 | หลัก | Key จาก 2 Citadel Maps ใน Fortress |
| Arbiter of Ash | หลัก | ย้ายมาใน Fortress |
| Xesht, We That Are One | Breach | Breachstone → Breach Domain → Tul+Esh → Key |
| King in the Mists | Ritual | Tribute → Audience → Rite of the Nameless → Key |
| Queen in the Mists 🆕 | Ritual | Atlas Tree node |
| Olroth | Expedition | Logbook → Ocean exploration → Islands |
| Vessel of Kulemak | Abyss | Abyssal Depths |
| Kosis | Delirium | Simulacrum (7 waves → Key) |
| Atziri | Vaal Temple | Temple completion |

## 10. ระบบ Runic Ward + Kalguuran Skills

### Runic Ward
- พลังชีวิตสำรอง — ทำงานเมื่อ Life เหลือ 1
- ฟื้นฟูแยกจาก Life
- เพิ่มให้ Armour ผ่าน Verisium Runeforging
- Armour Lv<55 → Runic Ward ไม่มี downside
- Armour Lv≥55 → แลก defences บางส่วนเป็น Runic Ward

### Kalguuran Skills (23 สกิลใหม่)
- ใช้ **Runic Ward** แทน Mana
- ไม่มีสี (colorless) — ใช้ได้ทุกคลาส
- ไม่ต้องใช้ Attribute
- ตัวอย่าง: Frostflame Nova, Rain of Blades, Skyfall, Grim Pillars
